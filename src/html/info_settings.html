Settings

Bitcoin Denomination: Change the view of bitcoin values throughout the app. (Default: bits)

Account: Change your account password, withdrawal PIN or recovery questions and answers.

Name: Optionally enter your first name, last name, and/or nickname. These fields will be sent to users you transact with. This information is never known by [[abtag APP_TITLE]] or any parties other than those you transact with.

Auto log off after: After the app has been backgrounded, amount of time before the user should be auto-logged out.
Default Currency: Change the default fiat currency for new wallets and for the total balance in your Wallets list. Note that this does not change the fiat currency used for transactions in individual wallets. Create new wallets to utilize different fiat currencies

Change Categories: Customize the list of transaction categories that appear in Transaction Details.

Spending Limits: Set Daily (Password protected) and Per-transaction (PIN protected) spending limits in Send Confirmation.

2 Factor: Enable 2 Factor Authentication to provide enhanced security.

Merchant Mode: Enable for merchant friendly transaction flow. After payment is received, app will return to Request screen instead of being prompted for transaction details.

Bluetooth: Enable to allow requesting/sending of bitcoin from someone in your vicinity via Bluetooth Smart (BLE).

PIN Re-Login: Enable to allow a log in with the 4 digit PIN instead of the full account password.

Use TouchID: Enable to allow login using TouchID fingerprint reader on supported devices.

Default Exchange: Choose the exchange service to use to determine the exchange rate of bitcoin with various fiat currencies.

Override Bitcoin Servers: Specify trusted bitcoin nodes to connect to for blockchain interaction. Nodes must be running Electrum servers
